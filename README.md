## FT Newton

Basic physics engine (like PhysX, Havok, or
Bullet Physics Engine)

[More information about project in subject file.](https://cdn.intra.42.fr/pdf/pdf/60852/en.subject.pdf)

Install Simple DirectMedia Layer

```shell
./install_SDL.sh
```

Compile project

```shell
make all
```

### Usage

Run

```shell
./ft_newton
```

Control
```
W - Move forward
A - Move left
D - Move Right
S - Move backward

Space - Move up
Shift - Move down

Left Click - Shoot
Right Click - Wind

1 - Spawn balls
2 - Spawn walls
0 - Clear all structs

Y - Increase balls for shoot
U - Increase balls size

< - Decrease gravity
> - Increase gravity
/ - Default gravity

ESC - Exit
```
