## FT Newton

Basic physics engine (like PhysX, Havok, or
Bullet Physics Engine)

[More information about project in subject file.](https://cdn.intra.42.fr/pdf/pdf/60852/en.subject.pdf)

Install Simple DirectMedia Layer

```shell
./install_SDL.sh
```

Compile project

```shell
make all
```

### Usage

```shell
./ft_newton
```
